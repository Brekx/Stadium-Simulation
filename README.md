# Projekt symulacji stadionu :stadium:
Symulację wywołuje się poprzez automatyczne zbudowanie symulacji poleceniem 
```
./gradew run
```
 a dokumentacja jest generowana w build/docs/index.html po wydaniu polecenia 
 ```
 ./gradlew javadoc
 ```
